## 说明
本人实测使用过的字库有simsun12.fun，HZK12，HZK16s，HZK24S，该HZK24S就是font文件夹下的

**数字是像素值，12 是 12*12，16 是 16*16，24 是 24*24；**

**数字后面的字母代表着字体，S 是宋体，F 是仿宋，K 是楷体，H 是黑体；**

```

HZK24F	HZK24H	HZK24K	HZK24S

HZK48
```
